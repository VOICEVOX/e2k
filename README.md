# e2k

英単語から読みを推測するライブラリ。\
関連Issue：[VOICEVOX/voicevox_project#65](https://github.com/VOICEVOX/voicevox_project/issues/65)

## リポジトリ構造

このリポジトリは以下の構造になっています。

- `infer/`：読みを推測するためのコード。
- `train/`：モデルを学習するためのコード。
- `dataset/`：データセットを生成するためのコード。
- `tools/`：その他、CI用のスクリプトなど。
